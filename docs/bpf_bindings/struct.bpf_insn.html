Berkeley Packet Filter Instruction

This is the general layout of a BPF Instruction. Not all fields are used on every instruction. Code defines what the instruction will do. JT and JF are used on branching instructions. Jump Forward, Jump Back. They are used to manipulate the PC register. K is a data value. It's purpose changes based on the code field.
